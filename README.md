# Simple Mobile Push API Demo

### Overview:

* Simple & ugly
* Authenticates with the API
* Lists all messages
* Deletes or Sends a single message

### Required Config Variables:

Create these using AppCenter on [Code@](https://code.exacttarget.com).

* FUEL_CLIENT_ID
* FUEL_CLIENT_SECRET
